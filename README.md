# IoT Telemetry Simulator in Modern C++20

This project is a small embedded/IoT-style telemetry simulator written in Modern C++20.

It practices:

- CMake
- source/header separation
- namespaces
- functions
- enum class
- std::string
- std::array
- std::vector
- const and constexpr
- control flow
- input validation
- formatted output
- RAII using std::ofstream

## Build

```bash
cmake --preset debug
cmake --build --preset debug
